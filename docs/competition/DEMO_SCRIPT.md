# Lumiq AI Demo Script

## Goal
- Show one believable loop: assessment -> personalized path -> tutor with context.
- Keep the demo to 3-5 minutes.

## Script
1. Open the landing page.
   - Say: "Lumiq AI giúp học sinh lớp 10 biết mình yếu gì để học đúng chỗ, thay vì học lại cả chương."
2. Start the assessment.
   - Say: "Prototype này chỉ dùng 6 câu để chẩn đoán nhanh 3 competency quan trọng."
3. Complete the assessment and show the result screen.
   - Say: "Sau assessment, hệ thống xác định đúng competency yếu nhất thay vì cho kết quả chung chung."
4. Enter the personalized learning path.
   - Say: "Current goal đổi theo kết quả vừa rồi. Đây là bằng chứng chính cho personalization."
5. Switch between scenario A and scenario B.
   - Say: "Nếu Minh yếu phép toán tập hợp thì current goal là M10.2. Nếu Minh yếu hàm bậc hai thì current goal nhảy sang M10.3."
6. Ask the tutor a question about the active topic.
   - Example: "Em không biết bắt đầu từ đâu với phần này."
   - Say: "Tutor không trả lời generic, mà bám vào topic đang học và điểm yếu hiện tại."
7. Close with the product promise.
   - Say: "Bản hiện tại tập trung vào Toán lớp 10, nhưng kiến trúc này có thể mở rộng sang môn và nhóm người học khác."

## Backup lines
- If the API is slow: "Prototype có fallback context để giữ trải nghiệm demo ổn định."
- If asked why the scope is narrow: "Chúng tôi cố ý làm hẹp để chứng minh một loop cá nhân hóa thật, không dựng slide đẹp nhưng flow rỗng."
